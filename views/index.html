Colored Text Api

Colored text api

A simple api to return rendered images based on the params givem to the url !!

Params:

  • text - pass the text content (required)
  • color - pass the text color hex rgba ex: FF0000FF (optional)
  • margin - pass the margin between borders ex: 30 (optional)
  • font_size - pass the margin between borders ex: 30 (optional)
the route is in /generate , the params is passed by query.


Example text only
https://gustavocodigo-colortext.glitch.me/generate?text=Im%20a%20cute%20butterfly


Example color
https://gustavocodigo-colortext.glitch.me/generate?text=hey&color=00ff00


Example font size
https://gustavocodigo-colortext.glitch.me/generate?text=hey&color=ed5ad7&font_size=50


the api will return a png.
